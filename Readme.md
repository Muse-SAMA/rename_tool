#  Windows 文件批量重命名小工具

语言：python 3.10.8

总体目标：开发一个 Windows 桌面小程序，支持用户通过图形界面选择多个文件，执行批量重命名操作，提供自定义的命名规则，并避免重名冲突。

核心功能：

1. 文件选择模块

(1) 支持通过文件选择窗口选择多个文件

(2) 支持直接框选多个文件并且通过右键菜单打开该工具进行重命名

(3) 支持拖拽文件到窗口添加

(4) 展示已选文件列表，显示原始文件名和路径

(5) 支持清除已选列表或重新选择文件

(6) 防止乱码、支持非英文文件名

2. 重命名规则设置模块
3. 冲突检测与处理模块
4. 预览模块
5. 执行重命名模块
6. 日志记录模块（可选）
